# 项目概览：tortoiseV2 (乌龟养护记录)

## 项目简介
这是一个专注于乌龟养护记录的微信小程序，旨在帮助用户记录和管理乌龟的日常饲养、状态及相关信息。

## 技术栈
- **前端框架**：uni-app
- **UI 组件库**：uni-ui
- **语法标准**：Vue 3
- **后端服务**：微信云开发 (WeChat Cloud Development)

## 主题配色方案 (Theme Colors)
项目采用自然清新的绿色系作为主色调：
- 浅绿：`#DBE9B2`
- 偏绿：`#C8E39A`
- 中绿：`#B0DB7B`
- 主绿：`#6CC42C`
- 深绿：`#65B028`

> 建议：在开发过程中，可以将这些颜色定义为 CSS 变量或在 `uni.scss` 中配置为全局 SCSS 变量，以便统一管理和使用。

## 后端架构与路由
- **后端入口文件**：`/api/index.js`
- **路由管理**：采用 `tcb-router` 进⾏云函数路由分发。
- **目录说明**：后端代码现统一存放在根目录 `/api` 下。
- **核心模块**（基于目前的路由配置推断）：
  - `user`: 用户管理相关
  - `pet`: 宠物（乌龟）信息管理
  - `class`: 分类或种类管理
  - `record`: 养护记录（如喂食、换水、体重、健康状况等）

## 记忆与规则
- **文件读取限制**：从现在开始，只读取用户明确 @ 的页面，其他页面一律不读取。

## 开发规范 (Best Practices)
- **云函数 Event 获取**：
  由于 `tcb-router` 或云函数环境的差异，`ctx` 可能被包裹了一层。为了兼容性，所有控制器中获取 `event` 参数统一使用以下写法：
  ```javascript
  const event = (ctx._req && ctx._req.event) ? ctx._req.event : (ctx.event || ctx);
  ```
